# IEEE CASS Kerala Chapter Website

This repository contains the code for a responsive website created for the **IEEE Circuits and Systems Society (CASS) – Kerala Chapter**. The site presents information about the chapter, its activities, leadership team, events, and contact details.

The website includes a homepage with animated statistics, sections for about and events, a team page with interactive member cards, and a contact page with basic communication information.

## Technologies Used

The site was built using **HTML, CSS, and JavaScript**, with custom styling based on the IEEE CASS color palette and the **Poppins** font. Interactive features such as the splash screen, animated counters, and team pop-ups were implemented using vanilla JavaScript.

The project is hosted using **GitHub Pages** for easy deployment and updates.
